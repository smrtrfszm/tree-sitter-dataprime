=======
extract
=======

extract message into my_data using regexp(e=/regexp/)

---

(query
  (extract_command
    expr: (key)
    into: (key)
    extract_function: (extract_function
      type: (identifier)
      (extract_argument
        argument: (identifier)
        value: (regex
          (regex_pattern))))))

=========================
extract multiple argument
=========================

extract message into my_data using regexp(e=/regexp/, b='asd')

---

(query
  (extract_command
    expr: (key)
    into: (key)
    extract_function: (extract_function
      type: (identifier)
        (extract_argument
          argument: (identifier)
          value: (regex
            (regex_pattern)))
        (extract_argument
          argument: (identifier)
          value: (string)))))
